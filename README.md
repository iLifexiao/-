## 使用
- 第一次使用需要设置唯一标识的6位数密码
- 用于记录、分类用户的帐号密码
- 可以随机生成密码
- 常用的编码加密
- 帐号、明文的加密
- 可以导入、导出加密后的帐号为手机联系人
